1. 以上四门课程，各需要半天时间。
1. 《前端开发的历史和趋势》适合所有听众，包括工程师、产品经理、管理人员等等。
1. 《React 技术栈》适合专业的前端开发者。
1. 《Node 应用开发》和《前端工程简介》适合使用 Node 的全栈工程师。
